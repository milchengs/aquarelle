Aquarelle Design System

by Emily Cheng

About

This website provides documentation for the Aquarelle Design System that I made for my portfolio. The name "Aquarelle", which is a method of watercolor painting, was inspired by the cool and fresh color palette with predominantly light colors. The keywords I wanted to incorporate in the design system are: clean, simple, and friendly.

Foundations

This section covers the design foundations for the design system: typographical hierarchy, color palette, and logo variations.

Components

This section covers reusable components needed to build the portfolio website, including navigation bars, footers, and cards.
